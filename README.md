## 使用機材
### PC
Window, Mac, Linuxどれでも大丈夫だが、Python3が動く環境であること
### ドローン
使用するドローンはTello
Amazon等で13000円前後で購入でき、スマートフォンのアプリでラジコンのように動かすことが出来る。  
重さが80gのため、現在のそのため、「航空法」の対象外になります。  
とは言え、気をつけなければならないので、所々確認して飛ばしましょう。
